About this section

This section is covering all the UI Elements used in PingOne web portal.

Status

We are permanently adding new elements as far as the development of the app is demanding, so this is a work in progress.

Labels and copy

In this static HTML template, we tried to keep data agnostic so we used dummy text whenever possible. When necessary, we used copy that would describe the UI Element we referred.

Branding

The elements used here, are going to be screenshot one by one and used for reference in the UI Elements Repository document, shared with the branding team.
They will have the chance to provide their branded version when necessarily, in that document. Each time when this will happen, Lucian will create the css and update it there so can be checked in the web environment, rather than on a bitmap image.
Compared with a static bitmap/vector image, the web environment is displaying all states of the UI elements (hover, selected, etc.), so this is the preferable way to work around this branding/UI aspect.

Contact / Project Owner

For any question about it, please check with Lucian.
